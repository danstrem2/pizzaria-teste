Passo a Passo Finaaaal:

Na VPS (onde o túnel está rodando): Aperte Ctrl + C (para parar o atual).

Rode este comando para iniciar pelo PM2:

pm2 start "cloudflared tunnel --url http://localhost:3001" --name tunnel
Salve essa configuração para voltar se reiniciar a VPS:

pm2 save
Descubra o Novo Link:

pm2 logs tunnel --lines 20
